Informativa sulla Privacy per @antibot-global

Ultimo aggiornamento: Maggio 2026

1. Informazioni generali

Questa applicazione ("@antibot-global") è uno strumento automatizzato personale sviluppato per identificare e moderare i commenti di spam/bot su YouTube.

2. Dati raccolti e utilizzo delle API di Google

L'applicazione utilizza i servizi API di YouTube per accedere ai commenti del canale dell'utente autenticato. I dati letti vengono utilizzati esclusivamente per verificare la presenza di tag di segnalazione ed eseguire risposte automatiche.

3. Conservazione dei dati

Nessun dato personale o credenziale viene inviato a server esterni o a terze parti. Il conteggio delle segnalazioni dei bot e i token di autenticazione vengono salvati localmente sul dispositivo dell'utente (tramite database SQLite locale e file token.json).

4. Contatti

Per qualsiasi domanda o richiesta di feedback, è possibile contattare lo sviluppatore all'indirizzo email: antibotspam.bot@gmail.com
